## 2. 节点操作

### 2.1 创建节点

| 方式 | 操作 |
|------|------|
| **手动添加** | 点击工具栏 **「添加节点」** 按钮，节点出现在画布中央 |
| **AI 节点** | 点击工具栏 **「AI节点」** 按钮，创建带 AI 处理能力的节点 |
| **文本自动拆分** | 在输入框粘贴文字 → 点击 **「自动转化」**，自动生成逻辑链节点组 |

---

### 2.2 移动节点

- **拖拽**：按住节点任意位置拖动（注意：不要拖到标题或内容编辑区）
- 节点会在画布内自由定位，不限制边界

---

### 2.3 编辑节点内容

**方式一：直接点击编辑（推荐）**

| 区域 | 操作 |
|------|------|
| **标题** | 单击标题文字，直接原位编辑 |
| **内容** | 单击内容文字，直接原位编辑 |
| **退出编辑** | 点击节点外部任意位置，自动保存 |
| **取消修改** | 编辑中按 `Escape`，恢复上次内容 |
| **标题换行限制** | 标题区按 `Enter` 结束编辑（不换行） |

**方式二：双击打开编辑弹窗**

双击节点空白区域，弹出完整编辑面板，可以：
- 修改标题和内容
- 调整节点**宽度**（拖动滑块，140px ~ 480px）
- 勾选 **「标记为已完成」**
- 点击 **「删除节点」** 删除该节点

---

### 2.4 调整节点大小

- 拖拽节点**右下角**的拖拽手柄（小方块）可以自由调整节点尺寸

---

### 2.5 标记已完成

| 方式 | 操作 |
|------|------|
| **快捷勾选** | 节点左下角有复选框 **「✓ 已完成」**，直接勾选 |
| **弹窗设置** | 双击节点 → 勾选「标记为已完成」→ 保存 |

**已完成节点外观：**
- 顶部显示绿色细横条（约 5px）
- 背景呈淡绿色渐变
- 标题文字显示删除线
- 内容区域透明度降低，视觉上归档感强

---

### 2.6 删除节点

| 方式 | 操作 |
|------|------|
| **键盘** | 选中节点后按 `Delete` 或 `Backspace` |
| **弹窗** | 双击节点 → 点击「删除节点」按钮 |

> ⚠️ 删除节点时，与该节点相关的所有连线也会一并删除

## 3. 连线操作

### 3.1 创建连线

1. 将鼠标悬停到**节点边缘**，出现蓝色圆点（**连接点 Pin**）
2. 从一个节点的 Pin **拖拽**到另一个节点的 Pin
3. 松开鼠标，连线自动生成（带箭头方向）

> 💡 连接点位于节点四个边的中心，上/下/左/右各一个

---

### 3.2 连线标签

- 在连线上**双击**，弹出输入框，可以为连线添加文字标签
- 输入完成后按 `Enter` 确认，按 `Escape` 取消
- 标签显示在连线中央

---

### 3.3 删除连线

| 方式 | 操作 |
|------|------|
| **右键菜单** | 右键点击连线 → 选择「删除连线」 |
| **键盘** | 点击选中连线（变蓝）→ 按 `Delete` 键 |

## 4. 富文本编辑

在节点标题或内容区域**选中文字**后，会自动弹出浮动格式工具栏。

### 4.1 格式工具栏

| 按钮 | 效果 | 快捷键 |
|------|------|--------|
| **B** | 加粗 | `Ctrl + B` |
| *I* | 斜体 | `Ctrl + I` |
| <u>U</u> | 下划线 | `Ctrl + U` |
| ~~S~~ | 删除线 | — |
| **A（颜色）** | 设置文字颜色 | — |
| **✕** | 清除所有格式 | — |

### 4.2 使用方法

1. 单击节点进入编辑模式
2. 用鼠标**选中**想要格式化的文字
3. 浮动工具栏自动出现在选区上方
4. 点击对应按钮应用格式
5. 点击节点外部退出编辑，格式自动保存

> 💡 工具栏使用 `mousedown` 事件，不会因点击按钮而丢失文字选区

## 5. AI 自动生成

### 5.1 文本自动拆分（本地模式）

适合快速将一段文字拆解成逻辑节点链：

1. 在工具栏文本框粘贴或输入文字
2. 模型选择保持 **「📝 本地拆分」**
3. 点击 **「自动转化」**
4. 系统自动按标点、句子拆分文字，生成横向排列的节点链

---

### 5.2 AI 大模型生成

支持多种 AI 模型进行智能拆分和结构化：

**支持的模型：**
- OpenAI：GPT-4o、GPT-4o mini
- 通义千问：Qwen-Plus、Qwen-Turbo、Qwen-Max
- Google Gemini：2.0 Flash、1.5 Pro、1.5 Flash

**使用步骤：**
1. 在模型下拉框选择想用的 AI 模型
2. 在 **API Key** 输入框粘贴对应的密钥
3. 在文本框输入要处理的内容
4. 点击 **「自动转化」**
5. AI 会智能分析文本，生成带逻辑层级的节点图

---

### 5.3 AI 处理节点

单个节点也可以调用 AI 进行处理：

1. 点击工具栏 **「AI节点」** 或双击现有 AI 节点
2. 填写「提示词」（告诉 AI 怎么处理）
3. 填写「输入数据」（要处理的内容）
4. 填写 API 密钥和端点地址
5. 点击 **「执行处理」**，结果显示在下方
6. 确认满意后点击「保存」

## 6. 笔记节点

笔记节点是一种**轻量便签**，适合在画布上记录备注说明。

### 6.1 创建笔记

点击工具栏 **「笔记」** 按钮，画布上出现黄色便签样式的节点

### 6.2 编辑笔记

- **单击**笔记节点：进入编辑模式，可以直接输入 Markdown 文本
- **点击外部**：退出编辑，Markdown 自动渲染为格式化文字
- **双击**：重新进入编辑

### 6.3 笔记支持的 Markdown 语法

```markdown
# 一级标题
## 二级标题

**加粗文字**
*斜体文字*

- 无序列表项
- 列表项 2

1. 有序列表
2. 第二项

`代码片段`

> 引用文字
```

### 6.4 连接 Pin 控制

- 右键笔记节点 → **「显示/隐藏连接 Pin」**
- 显示 Pin 后可以像普通节点一样连线

## 7. 图片节点

支持将图片拖入画布，作为图片节点显示。

### 7.1 添加图片

- 将本地图片文件**直接拖拽到画布**上，松开即可创建图片节点

### 7.2 操作图片节点

| 操作 | 说明 |
|------|------|
| **拖动** | 拖拽图片节点可移动位置 |
| **调整大小** | 拖拽右下角手柄调整尺寸 |
| **删除** | 选中后按 `Delete` 键 |

> 支持 JPG、PNG、GIF、WebP 等常见格式

---

## 8. 分组功能

可以将多个节点框选后归为一组，方便管理大型图表。

### 8.1 创建分组

1. 在画布空白处**按住鼠标左键拖拽**，出现蓝色选择框
2. 框选多个节点后**松开**，弹出分组操作面板
3. 在面板中输入分组名称
4. 点击 **「创建分组」** 按钮

### 8.2 分组外观

- 选中节点会被一个带标签的**虚线矩形框**包围
- 分组标签显示在框的左上角

### 8.3 编辑分组名称

- **双击**分组标签文字，进入编辑，按 `Enter` 确认

### 8.4 取消分组

- 点击面板中的 **「取消选择」** 按钮

## 9. 配色定制

### 9.1 打开配色面板

点击工具栏 **「🎨 配色」** 按钮，右侧弹出配色面板

### 9.2 画布背景色

- 点击画布背景颜色色块，选择整个画布的底色

### 9.3 全局节点颜色

- 设置所有节点的**默认背景色**和**头部颜色**
- 影响全局，单独设置的节点颜色优先级更高

### 9.4 单个节点颜色

**右键**任意节点（非笔记节点），弹出颜色菜单：

1. **点击色板**：实时预览该颜色（节点即时变色）
2. **自定义颜色**：使用颜色拾取器，拖动时实时预览
3. **确认应用**：点击底部 **「✓ 确认应用」** 按钮，颜色正式保存
4. **取消**：点击「取消」或点击菜单外部，自动回滚到原色
5. **重置为默认**：清除单独设置，恢复全局颜色

> 💡 文字颜色会根据背景亮度自动切换为深色或浅色，保证可读性

### 9.5 恢复默认配色

配色面板底部有 **「恢复默认」** 按钮，一键重置所有颜色

## 11. 文件管理

### 11.1 保存文件

点击工具栏 **「💾 保存」** 按钮：
- 自动下载一个 `.json` 文件到本地
- 文件名格式：`blueprint_YYYY-MM-DD.json`
- 文件包含所有节点、连线、位置和颜色信息

### 11.2 打开文件

点击工具栏 **「📂 打开」** 按钮：
- 选择之前保存的 `.json` 文件
- 画布内容会被替换为文件中的内容

### 11.3 清空画布

点击工具栏 **「🗑 清空」** 按钮：
- 弹出确认对话框
- 确认后清除所有节点和连线

> ⚠️ 清空操作**不可撤销**，请先保存文件再执行

### 11.4 数据自动存储

- 每次操作后，数据自动保存到浏览器 `localStorage`
- 关闭页面后重新打开，数据依然保留
- **不同电脑/浏览器之间数据不同步**，需手动保存 `.json` 文件传输

## 12. 快捷键速查

### 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Z` | **撤销**上一步操作 |
| `Delete` | 删除**选中的节点**或**选中的连线** |
| `Backspace` | 同 Delete（在非输入框状态下） |
| `Escape` | 取消当前编辑（恢复原内容） |

### 编辑中快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + B` | 加粗选中文字 |
| `Ctrl + I` | 斜体选中文字 |
| `Ctrl + U` | 下划线选中文字 |
| `Enter`（标题中） | 结束标题编辑 |
| `Escape`（编辑中） | 取消修改，恢复原内容 |

### 连线标签快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 确认连线标签输入 |
| `Escape` | 取消连线标签输入 |

## 💡 使用技巧

1. **快速整理文字**：把会议记录、需求文档粘贴到输入框 → 自动转化，立刻得到可视化节点图

2. **颜色区分优先级**：用红色节点表示高优先级，绿色表示已完成，黄色表示待讨论

3. **分组管理大型图**：超过 20 个节点时，建议按模块分组，便于折叠和管理

4. **定期保存**：`localStorage` 有容量限制（约 5MB），大型项目建议定期导出 `.json` 备份

5. **撤销保护**：所有操作（添加/删除/移动/颜色）均支持 `Ctrl+Z` 撤销，放心操作

6. **颜色预览安全**：修改节点颜色时，点击「取消」或菜单外部会自动回滚，不用担心误操作

## ❓ 常见问题

| 问题 | 解决方法 |
|------|---------|
| 数据丢失了 | 检查是否清除了浏览器缓存；建议定期用「保存」导出 `.json` |
| 节点文字看不清 | 右键节点 → 更改背景颜色，文字会自动适配亮/暗 |
| 连线拖不出来 | 确保鼠标悬停在节点**边缘的蓝色圆点**上再拖拽 |
| AI 生成失败 | 检查 API Key 是否正确；检查网络是否能访问对应 AI 服务 |
| 画布乱了找不到节点 | 缩小缩放比例到 20%，查看全局；或使用「清空」重新开始 |
| 笔记显示源码 | 正常现象，单击笔记进入编辑模式，点击外部退出后自动渲染 |
